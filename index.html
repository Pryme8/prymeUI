<select> Supports:
  • Name
    Passes the element name from the original object
  • ID
    Passes the element id from the original object
  • Value
    Passes the original value of the element and makes the prymeElement set to that value
  • autofocus
    Checks for the attribute autofocus set in the original Element.
  • disabled
    Checks to see if the element is disabled and if so adds the class 'disable' to the prymeElement.
    ---I will be looking to add a listener for when ever this elements original is changed with script to be disabled or enabled that the prymeElement will follow suit.
  • form
    Because the original element is just hidden this should in theory still work... has not been tested
  • multiple
  • required
    Because the original element is just hidden this should in theory still work... has not been tested
